MCPS Sidekick

April 2019

MCPS Sidekick is a grade-viewer app I created and published in my junior year of high school. It originally used web requests to imitate to a server that it was a user logging in via a web browser. When MCPS changed to use a platform called StudentVue, I used their API and created a Dart package for others to use, which I left to a friend to update when I graduated high school. The app allows users to not only view their grades, but to modify scores on assignments to see what their grade would be if they received a different score on an assignment, as well as to add future assignments and input a score for them (ex. to see what grade they would need to get on a future quiz in order to get an A in a class). The project is no longer on the app store, and I am not maintaining it anymore as I'm not a user anymore. It is written in Flutter and released for iOS and Android. The GitHub repository is available here.
